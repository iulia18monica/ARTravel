# ARTravel - Cluj-Napoca City Guide & Trip Planner 📍

ARTravel is a comprehensive web application built with **Django** designed to help tourists and locals explore Cluj-Napoca. The platform offers a curated list of locations, user-generated reviews, and a smart route planner integrated with Google Maps.

## 🌟 Key Features

### For Visitors (Non-logged users):
* **Explore Categories:** Browse through various locations categorized by *Restaurants*, *Entertainment*, and *Self-care*.
* **Read Reviews:** Access feedback and ratings from other community members to make informed decisions.

### For Registered Users:
* **Personalized Favorites:** Add or remove locations from a private "Favorites" list.
* **Review System:** Share experiences by leaving reviews on visited locations.
* **Smart Route Planner:** * Add specific locations to a "To Visit" list.
    * Create and save custom routes for future trips.
    * **Google Maps Integration:** A dedicated "View Route" button that automatically generates a Google Maps link with all selected stops pre-configured for navigation.

### For Administrators:
* **Full Content Management:** Complete control over adding, editing, or deleting locations.
* **User Management:** Oversee user accounts and platform activity through the Django Admin dashboard.

## 🛠️ Technical Stack
* **Backend:** Python, Django Framework
* **Database:** SQLite (Development)
* **Frontend:** HTML5, CSS3 (Custom Styling)
* **Integration:** Google Maps URL API
